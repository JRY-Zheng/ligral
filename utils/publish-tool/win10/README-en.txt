Ligral (v0.2.2) installer for Windows 10

It's recommended to run this with admin's privilege.

This installer firstly unpacked itself to a temp dir and then install ligral to the default dir. If you want to self-defined the installation destination, modified the unpacking temp dir and choose don't install to the default dir in the cmd window while installing.
